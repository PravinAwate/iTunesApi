# iTunesApi

Clone the repo

  cd iTunes-Server

  Install dependencies.
  yarn install

  Build the  server.
  yarn build

  Run the server.
  yarn serve

-----------------------------------------------
Start UI

  cd itunes-ui

  Install dependencies.
  yarn install

  Build the  server.
  yarn build

  Run the server.
  yarn serve

  enter http://localhost:8080/#/ in browser
